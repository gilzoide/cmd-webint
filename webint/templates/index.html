Welcome!

This is Coh-Metrix-Dementia

Coh-Metrix-Dementia is a system that can analyze texts in terms of several metrics, which are related to cohesion, coherence, and readability, among other textual aspects.

WARNING: This is a pre-alpha version of the system, so more likely than not it contains lots of bugs. This version was released for internal testing purposes only. Please don't use the results of this system for any serious purpose before its final release.
