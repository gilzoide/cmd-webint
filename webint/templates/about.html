Coh-Metrix-Dementia 1.0

Coh-Metrix-Dementia is a tool that uses Natural Language Processing (NLP) tools and resources to allow for the extraction of textual metrics, which can be used to aid the automated analysis and detection of dementia, with initial focus on Alzheimer's Disease and Mild Cognitive Impairment. It's based on Coh-Metrix-Port, the Brazilian Portuguese adaptation of Coh-Metrix. Coh-Metrix-Dementia is capable of extracting all the 48 metrics originally supported by Coh-Metrix-Port, and also 25 additional metrics that capture syntactical complexity, idea density, textual coherence, and disfluency level. All supported metrics are described in the Help page.
